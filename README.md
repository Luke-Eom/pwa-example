## 1. 소개
- 이 곳은 **'Do it! 프로그레시브 웹앱 만들기'** (이지스퍼블리싱, 김응석 저) 책 **실습예제 완성 소스코드 원격저장소(Repository)** 입니다.
- 버전에 상관 없이 실행할 수 있는 `완성된 소스파일`과 책 내용에 따라 직접 따라해 볼 수 있는 `프로젝트 템플릿`이 준비되어 있습니다.

## 2. 사용 도구
- 소스코드 편집기 : `비주얼스튜디오 코드`

<br>

## 3. 사용방법
- 폴더 이름이 `ex01-01`은 `1장의 첫번째 예제`라는 의미입니다. 완성된 소스코드가 들어 있습니다.  
- 폴더 이름이 `ex05-07_start`는 `시작할 때 활용하는 프로젝트 템플릿`을 의미합니다. start 폴더를 열어 책 설명에 맞춰서 소스코드를 입력하며 학습하시면 됩니다.
